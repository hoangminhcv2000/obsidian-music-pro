# Changelog

## 1.0.0 - 2026-06-02

- Initial public release candidate.
- Added a 1000-playlist bundled SoundCloud catalog with duplicate playlist hiding and refreshed album artwork.
- Added Editor's Choice, Recent, personal playlist folders, quick pick, and compact mini dock.
- Added album-level artwork consistency in Recent while keeping track artwork available in the player.
- Added SoundCloud playlist hydration, short-track filtering, blocked/unavailable track handling, broken-link checks, and catalog audit scripts.
- Added performance optimizations for playlist switching, mini/full mode changes, category filtering, drag sorting, and hot progress updates.
