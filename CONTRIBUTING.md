# Contributing

Thanks for helping improve Music Pro.

## Report bugs

Open an issue with:

- Obsidian version.
- Music Pro version.
- Steps to reproduce.
- Screenshot or screen recording if the issue is visual.

## Suggest playlists

Open an issue with the SoundCloud link and a short note about where it fits best.

## Development

```bash
npm ci
npm run check
```

Before a release, run:

```bash
npm run release:prepare
```
